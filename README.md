# China Trip 2026 🇨🇳

Interactive travel itinerary for 16-day China adventure.

**Live site:** [aman-ankur.github.io/china-trip-2026](https://aman-ankur.github.io/china-trip-2026/)

## Trip Overview

| | |
|---|---|
| **Dates** | October 15-31, 2026 |
| **Duration** | 16 days |
| **Cities** | Guangzhou → Yangshuo → Chengdu → Jiuzhaigou → Chongqing → Zhangjiajie → Shenzhen |
| **Travelers** | 2 |

## Highlights

- 🐼 **Chengdu** — Giant Panda Research Base
- ⛰️ **Yangshuo** — Li River bamboo rafting & karst mountains
- 🏔️ **Jiuzhaigou** — Turquoise lakes & autumn foliage
- 🌃 **Chongqing** — Cyberpunk cityscape & Wulong Karst day trip
- 🎬 **Zhangjiajie** — Avatar's floating mountains & Tianmen Mountain
- 🏯 **Fenghuang/Furong** — Ancient towns with waterfall views

## Features

- Sticky navigation with anchor links
- Detailed day-by-day itinerary with HSR train times
- Hotel recommendations for each city
- Shopping guide (electronics, fashion, tea)
- Food highlights section
- Budget breakdown
- Mobile responsive design

## Tech

Single-page static HTML with:
- Modern CSS (Space Grotesk + DM Sans fonts)
- Pexels images (free, reliable CDN)
- No JavaScript dependencies
- GitHub Pages hosting

## Local Development

Just open `index.html` in a browser. No build step required.
